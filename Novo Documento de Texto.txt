Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass (temporario)
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned (permanente)

----------------------- INICIANDO -----------------------
1 - mkdir siteProjeto
2 - python -m venv dsweb
3 - .\dsweb\Scripts\activate
4 - pip install django; pip list
5 - django-admin startproject projeto_nome // dir raiz + manage.py
6 - cd mysite; python manage.py runserver // "roda o servidor"
7 - python manage.py startapp app_nome // cria uma app

----------------------- CRIANDO ESTILO -----------------------
1 - criar as pastas css, images e js no app
2 - criar no settings.py ==> STATIC_ROOT= os.path.join(BASE_DIR, 'staticfiles') # usado na hora da produção
3 - shell ==> manage.py collectstatic
            cria um dir na raiz do projeto com todos os arquivos estaticos
3 - servidor padrão não consegue servir arqvs estaticos qnd em modo de produção
